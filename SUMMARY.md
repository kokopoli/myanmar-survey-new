## 🚀 快速开始

### 1. 一键设置
```bash
# 运行快速设置脚本
pnpm run setup
```

### 2. 手动设置
```bash
# 安装依赖
pnpm install

# 创建环境变量
cp .env.example .env.local

# 初始化数据库
pnpm db:generate
pnpm db:push
pnpm db:seed

# 启动开发服务器
pnpm dev
```

### 3. 访问地址
- **调查问卷**: http://localhost:3000
- **CMS管理**: http://localhost:3000/admin
- **管理员账户**: admin@myanmar-survey.com / admin123
